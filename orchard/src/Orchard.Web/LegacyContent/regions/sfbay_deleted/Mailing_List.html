mailing list

Check out our mailing list at Yahoo!Groups. The service will allow us to inform everyone of events but also would allow for additional functions such as a group calendar planning for special interest events, polling, online invitations, etc. The only drawback is that you have to have a Yahoo ID in order to access the additional features. If you are apprehensive about setting up a Yahoo ID, you can use the universal Yahoo ID (sf-rice-alumni) with a password. Otherwise, like the listserv mail service, Yahoo!Groups will allow alumni to post emails to everyone that is subscribe to the service. Furthmore, sending a blank email to the mail server will allow you to subscribe and unsubscribe as shown below:

Post message: sf-rice-alumni@yahoogroups.com
Subscribe: sf-rice-alumni-subscribe@yahoogroups.com
Unsubscribe: sf-rice-alumni-unsubscribe@yahoogroups.com
