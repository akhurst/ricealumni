directory
Judging from the survey, most alumni have requested a directory. Because of security issues with posting the directory over the internet, the Alumni Association has not released the national database so that local Alumnus will be able to look each other up. I will post an update, once the security issues are resolved.

In the meantime, there is an unofficial directory for Rice Alumni where alumni can register and search for and keeping in touch with other long-lost alumni.


 mentoring
We are currently defining and developing a mentoring program for undergraduates and young alumni. We are looking for any suggestions on how to structure the program so that it is effective. We will be soliciting alumni for mentors as well as mentees. An alternative is to set up an informal networking list so that alumnus can contact each other for one-time or recurring advice.
